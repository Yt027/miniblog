I'm Yaya Tangara, and hope you are doing well.

This is a simple blogging website which aim to learn PHP basic functions to beginners and reinforce intermediate end even professional programmers.

In the development, Is used these technologies:
  • HTML - CSS - JavaScript
  • PHP
  • Font Awesome Icons

By rebuilding this website your own way, you'll learn to:
HTML: 
  1. Build customised textarea element, allowing you to add (underlined text, bold text, italic text, ordered and unordered lists, links and so more)
  2. Include Font Awesome Icons into your projects against adding multiple PNG or SVG icons into your server.
CSS:
  1. Learn display flex, flex properties and some of there values.
  2. Learn colour coherence and apply contrast into your elements.
  3. Make transitions on your input, button elements when they are focused or hovered.
JavaScript:
Learn to manage events on custom elements, even which are non-native HTML elements.
PHP:
  1. Learn how to automatically create new files on your server.
  2. How to delete server files.
  3. Make a root authentication process.
